Study · SQL for data analysis

SQL self-study syllabus

Foundations of SQL for data analysis — retrieval, filtering, functions, aggregation, joins, subqueries — ending with a documented portfolio project. Primary video track: Alex the Analyst.

SLOTS  Mon 16:45–17:30 + Sat 11:15–12:45
WEEKLY  ~2¼ h
GOAL  A multi-table portfolio query you can show the SE team

The four resources

  • Video (primary) — Alex the Analyst, MySQL series (in his free Data Analyst Bootcamp playlist on YouTube). One topic-titled video per week, listed by exact title.
  • Book — Sams Teach Yourself SQL in 10 Minutes (Ben Forta). Assigned chapters each week; ~10 min reads.
  • Practice — SQLBolt (sqlbolt.com), interactive in-browser lessons.
  • Sandbox — MySQL + Alex's sample database, plus read-only SIS queries at work where relevant.

Weekly rhythm

  • Mon (45 min) — read the book chapter(s), do the matching SQLBolt lesson(s).
  • Sat (90 min) — watch the Alex video, code along, then write 2–3 of your own queries against the sandbox.
  • Every week — log one useful query to the running cheat-sheet.
